# Skripted Project Guidelines

This project is a Next.js 16 application integrated with an Obsidian vault for knowledge management.

## Project Structure

- `src/`: Core application logic.
- `Skripted-Vault/`: Obsidian vault for personal knowledge management (PARA method).
- `public/`: Static assets and locales.

## Workspace Mandates

1.  **Vault Awareness**: For all non-coding tasks (research, planning, brainstorming), refer to `Skripted-Vault/GEMINI.md` for guidelines.
2.  **Surgical Code Updates**: Use the `replace` tool for targeted code changes.
3.  **Language**: The project supports English and Turkish (locales in `public/locales`).
4.  **Tech Stack**: Next.js 16, Supabase, Tailwind CSS, TypeScript.

## Key Files

- `Skripted-Vault/GEMINI.md`: Detailed instructions for the Obsidian vault.
- `package.json`: Project dependencies and scripts.
- `next.config.ts`: Next.js configuration.
